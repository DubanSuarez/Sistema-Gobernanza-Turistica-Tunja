# Sistema de Gobernanza Turística - Tunja

Sistema web desarrollado para la recolección, gestión y análisis de datos turísticos en la ciudad de Tunja.

## Objetivo
Apoyar la toma de decisiones mediante el análisis de información sobre visitantes, percepción, cultura y comportamiento turístico.

## Tecnologías
- PHP
- MySQL
- JavaScript
- HTML/CSS

## Funcionalidades
- Registro de turistas
- Análisis de percepción
- Gestión de gastos
- Visualización de datos

## Contexto
Proyecto académico enfocado en gobernanza turística y análisis de datos.
